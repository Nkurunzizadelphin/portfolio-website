# portfolio-website
My personal portfolio website
# Collaboration Workflow

## Steps Followed

1. Created a personal portfolio repository
2. Uploaded website files to GitHub
3. Created a separate feature branch
4. Added a new webpage (blog.html)
5. Committed changes to the branch
6. Created a pull request
7. Merged the branch into main
8. Deployed the website using GitHub Pages

## Tools Used

- GitHub
- Git
- GitHub Pages
- Branches
- Pull Requests
